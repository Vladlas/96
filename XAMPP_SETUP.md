# Запуск проекта в XAMPP (Apache + PHP)

Проект не требует Node.js, npm и сборки — это чистый PHP.

## 1) Копирование в XAMPP

Скопируйте файлы проекта в папку:

- `C:\xampp\htdocs\museum96\`

Минимально должны быть:

- `C:\xampp\htdocs\museum96\index.php`
- `C:\xampp\htdocs\museum96\data.php`
- `C:\xampp\htdocs\museum96\styles.css`
- `C:\xampp\htdocs\museum96\media\...` (ваши изображения)

## 2) Добавьте изображения

Положите медиа-файлы в:

- `C:\xampp\htdocs\museum96\media\`

Список нужных имён — в `public/media/README.md`.

## 3) Включите Apache в XAMPP

Запустите XAMPP Control Panel и включите **Apache**.

## 4) Откройте сайт

Откройте в браузере:

- `http://localhost/museum96/`

## 5) Важно по сохранению данных

Добавленные через админ-панель заметки и события хранятся в `$_SESSION`:

- они сохраняются в пределах текущей сессии браузера;
- после очистки cookies/сессии исчезнут.

## 6) Частые проблемы

1. **Не подгружаются изображения**
   - Проверьте имена файлов и путь `media/...`.
2. **Apache не запускается**
   - Проверьте, не занят ли порт `80` другим приложением.
3. **Открывается список файлов вместо сайта**
   - Убедитесь, что входная точка называется `index.php`.
