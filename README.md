# Виртуальный музей кафедры 96 (PHP + XAMPP)

Проект переписан с React на чистый PHP, чтобы запускаться в XAMPP без Node.js и сборки.

## Быстрый старт

1. Скопируйте проект в `C:\xampp\htdocs\museum96\`.
2. Запустите **Apache** в XAMPP.
3. Откройте `http://localhost/museum96/`.

## Что реализовано

- Навигация по залам (`Главный`, `История`, `Наука`, `Люди`).
- Галереи с fallback, если изображения отсутствуют.
- Быстрые маршруты экскурсии.
- Кураторские заметки с модальным окном.
- Админ-панель:
  - добавление заметок;
  - добавление событий календаря.

Данные админ-панели сохраняются в `$_SESSION` (до перезапуска сессии браузера).

## Структура

- `index.php` — основная страница и обработка форм.
- `data.php` — все данные музея.
- `styles.css` — стили интерфейса.
- `media/` — папка с изображениями (создайте в корне сайта при деплое).
